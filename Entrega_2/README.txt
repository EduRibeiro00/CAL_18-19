Lista de dependencias:
GraphViewer (em caso de Windows, biblioteca Wsock32)
Ficheiro com veiculos e central
Ficheiro com entregas
Ficheiros dados por monitor para a leitura de grafos

Instrucoes de compilacao:
Criar um novo projeto no Eclipse, e copiar as pastas
e ficheiros para dentro dele, com a estrutura de diretorios
intacta. Compilar e correr.
